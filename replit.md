# DOZEY (Community)

## Overview
A vaccination record tracking application built with React + Vite + Tailwind CSS v4. Helps users track vaccination compliance across countries, manage documents, and share records.

## Tech Stack
- React 18 with TypeScript
- Vite 6 (dev server on port 5000)
- Tailwind CSS v4 (via @tailwindcss/vite plugin)
- Shadcn/ui components (Radix UI primitives)
- Lucide React icons

## Project Structure
- `src/App.tsx` - Main app with state management and routing
- `src/components/` - Feature components (Dashboard, Profile, etc.)
- `src/components/ui/` - Reusable UI components (shadcn/ui)
- `src/styles/globals.css` - Theme variables and base styles
- `src/index.css` - Tailwind imports
- `vite.config.ts` - Vite configuration with Tailwind plugin

## Running
- `npm run dev` starts the development server on port 5000
